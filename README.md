# Bem vindo a apresentação do meu projeto

##

Pois bem neste código com a ajuda do bot Lovable eu pedi a ajuda para fazer um sistema, onde nele consta um Dashboard geral do gerenciamento de uma clinica de atendimento ao público.
No começo do código havia apenas o esqueleto, apenas a parte do Dashboard inicial, havia os botões laterais, porém nenhum funcionava, conforme fui pedindo ajuda e fazendo o código junto o site foi tomando um pouco de forma e os botões começaram a funcionar.
Neste dashboard, possui a aba "Dashboard" onde seria a parte principal contendo todas as informações; a aba "chat" onde é a parte de conversas para atendimento, marcação de horário, consultas, etc; aba "equipe" onde mostra os funcionarios/ atendentes, e nesta aba você pode vizualizar que está disponivel para atendimento.

Com a ajuda do bot, pude começar e aprimorar meu site para o "exemplo" de uso do site ao público, o site ainda não está 100%, ainda há muito a ser feito para deixar utilizavel
